Studio예제: 챠트 - 세로 / 가로 막대 그래프(2) Multi-bar chart
============================================================================

| 막대그래프(bar chart) 는 연속형 변수가 아닌 이산형 변수에 대해 범주(category), 계급(class)별로 빈도나 합계 등을 막대로 그려서 비교할 때 유용합니다.
| multi-bar chart 는 1개 범주나 class 에 2개 이상의 빈도나 합계 등을 막대로 그린 것입니다.

Multi - 세로 막대 그래프
